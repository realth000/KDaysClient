# kdays_client

Client for KDays

## 编译

配置[flutter](https://flutter.cn/)环境。

```shell
# 1. 代码生成
dart run build_runner build

# 2. 编译

## 安卓
flutter build apk

## iOS
flutter build ios --no-codesign

## Linux
flutter build linux

## MacOS
flutter build macos

## Web
flutter build web

## Windows
flutter build windows
```
